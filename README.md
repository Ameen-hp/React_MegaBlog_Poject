MegaBlog

A modern blogging platform built with React, Tailwind CSS, Redux, and Appwrite.
Users can create, format, and manage blog posts with a rich text editor (TinyMCE), featuring slug generation, post fetching, and deletion.

Features

Create Post – Write and publish blog posts with rich formatting.

Text Formatting – Powered by TinyMCE for headings, bold, italic, images, and more.

Slug Support – Auto-generated slugs for SEO-friendly post URLs.

View All Posts – Fetch and display posts from Appwrite Database.

Delete Post – Remove any post securely via Appwrite backend.

Global State – Managed using Redux Toolkit.

Responsive UI – Styled with Tailwind CSS for a modern look.

Tech Stack

Frontend: React, Tailwind CSS

State Management: Redux Toolkit

Backend & Database: Appwrite

Rich Text Editor: TinyMCE
